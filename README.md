# 🍫 어린이 초콜릿 체험 관리 웹페이지  

## 📌 프로젝트 개요  
이 프로젝트는 지역 초콜릿 제작 카페와 협업하여 제작된 **어린이 초콜릿 체험 관리 웹페이지**입니다.  
체험 현장을 관리하는 담당자가 아이들을 쉽게 통제하고, 체험 진행 과정을 효과적으로 안내할 수 있도록 돕기 위해 개발되었습니다.  

## 주요 기능  
- **체험 선택 기능**: 아이들이 몰드 초콜릿, 빼빼로 만들기 등 다양한 체험을 선택할 수 있음  
- **음성 안내 시스템**: 원숭이 캐릭터가 체험 순서를 음성으로 설명하여 아이들이 쉽게 따라할 수 있음  
- **대본 및 순서 편집 기능**: 담당자가 대본의 내용과 체험 진행 순서를 직접 수정할 수 있도록 드래그 앤 드롭 기능 제공  
- **체험 진행 시간 조정**: 모든 체험이 일정한 시간 내에 완료될 수 있도록 대본별 대기 시간을 설정 가능  

## 프로젝트 배경  
초콜릿 체험 프로그램을 운영하는 담당자는 체험에 참여하는 아이들을 쉽게 관리할 수 있는 시스템이 필요했습니다.  
기존에는 체험 진행 과정이 수기로 관리되었으며, 체험자의 집중도를 유지하기 어려운 문제가 있었습니다.  
이를 해결하기 위해 **직관적인 UI**와 **친근한 캐릭터**를 활용한 웹페이지를 제작하였으며,  
**자동 음성 안내 시스템**을 도입하여 체험을 보다 원활하게 진행할 수 있도록 하였습니다.  

## 해결한 문제  
✅ **비용 절감**: 실시간 TTS API 호출을 고려했으나, 호출 비용 문제로 인해 대본을 사전에 음성 파일로 변환하는 방식으로 개선  
✅ **직관적인 편집 인터페이스 제공**: 체험 진행 대본과 순서를 쉽게 변경할 수 있도록 드래그 앤 드롭 기능 추가  
✅ **체험 진행의 효율성 향상**: 체험 진행 시간을 조정할 수 있는 기능을 추가하여 모든 아이들이 동일한 시간 안에 체험을 마칠 수 있도록 설계  

## 프로젝트 결과  
본 웹페이지는 **Vercel을 통해 배포**되었으며, 실제 초콜릿 체험장에서 사용되었습니다.  
이를 통해 담당자는 아이들을 보다 쉽게 관리할 수 있었고, 체험을 진행하는 아이들도 **직관적인 UI**와 **음성 안내** 덕분에 체험을 더욱 즐겁게 경험할 수 있었습니다.
